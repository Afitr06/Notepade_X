# Notepad

Sebuah notepad ringan sederhana open-source.

## git

![git hub](https://github.com/Afitr06)

## Features

- Menambahkan catatan
- Sunting Catatan
- Hapus Catatan
- Hapus Banyak Catatan
- Hapus catatan dari sampah secara permanen
- Terapkan tag ke Catatan
- Tambahkan Label
- Sunting Label
- Hapus Label
- Hapus Beberapa Tag
